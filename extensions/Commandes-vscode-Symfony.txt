CTRL+Shift+P -> Preferences Open user setting
CTRL+SHIFT+X -> Extendions

Creer projet symfony: (ou ouvrir repertoire projet existant si deja cree)
cd <repertoire root>
composer create-project symfony/website-skeleton New_Project

Installer Server PHP pour remplacer Apache sur Env Local dev:
composer require server --dev

Demarer Server:
php bin/console server:run

Creer BDD:
php bin/console doctrine:database:create

Créer class Entity (wizard pour aide a la creation d'une class datamodel):
php bin/console make:entity

Créer class Migration (wizard pour aide a la creation d'une class migration):
php bin/console make:migration

Migration mettre a jour une BDD à partir des class Migration existantes:
php bin/console doctrine:migrations:migrate

Installer librairies Fixtures (tests en BDD):
composer require orm-fixtures --dev

créer une class Fixtures (wizard pour aide a la creation d'une class de creation de données de test en BDD):
php bin/console make:fixtures

Charger les Fixtures:
php bin/console doctrine:fixtures:load

Créer Formulaire (wizard pour aide a la creation d'une class formulaire):
php bin/console make:form
